About Us

Welcome to Recipe Finder! Our mission is to bring the joy of cooking to kitchens everywhere. We believe that anyone can create delicious meals, and our platform is designed to make finding and following recipes as easy and enjoyable as possible.

This project was built to demonstrate modern web technologies, including a Python Flask back-end that communicates with the Spoonacular API to fetch live recipe data, and a dynamic front-end built with HTML, CSS, and vanilla JavaScript.
